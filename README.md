# 🔗 Автоматическая редирект-страница

🌿 Простое решение для автоматического редиректа с анимационными эффектами.


## 🚀 Быстрый старт

1. Скопируйте этот репозиторий
2. Откройте файл `index.html`
3. Измените параметры (см. ниже)
4. Загрузите на свой хостинг

## Настройка редиректа

### Основные параметры (изменяются в HTML-файле):

```html
<!-- В head -->
<meta http-equiv="refresh" content="3; URL=https://ВАШ_САЙТ.ru/">

<!-- В body -->
<a href="https://ВАШ_САЙТ.ru/" class="btn">Перейти на сайт</a>
```

## Особенности
1. Полностью адаптивный дизайн
2. Плавные анимации загрузки
3. Оптимизирован для SEO
4. Поддержка всех современных браузеров

## Дополнительные возможности

**Добавьте логотип:**

```html
<div class="logo">
  <img src="your-logo.png" alt="Ваш логотип">
</div>
```
**Изменение времени редиректа:**

Отредактируйте значение в `content="3"` (в секундах)
